How to Use

Simple Compression

Choose an image or multiple images by clicking on "choose image or images" button, choose your images you want to compress. The output will be in the same folder where you have chosen your images and will have an extension _imagem8

Scale by Width or Height

Choose from dropdown how you want the scaleFactor to behave if you choose width, the tool automatically calculates the height so that the aspect ratio is contained

Quality Slider

The Quality slider can be used to change the quality of the image, For .png images, it is used to resize accordingly

Choose save folder

Allows you to store the resized/compressed images in separate folder, they will retain the same name

Force Jpg

This option forces images to be compressed to JPG format
