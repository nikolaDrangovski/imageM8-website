How to Use

Simple Compression

Choose an image or mupltiple images by clicking on the button "choose image or images", choose your images that you want to compress. The output will be in the same folder where you have chosen your images with an extension _imagem8

Custom Width and Height Compression

Choose custom from the dropdown, input your dimensions and choose image to resize

Eather width or height

Choose from dropdown whant how you want the scaleFactor to behave if you choose width the tool automaticly calculate the height so that the aspect ratio is contained and you don't loose the image look.

Quality slider

The Quality slider can be used to change the quality of the image, For .png images it's used to resize them acordingly.

Choose save folder

Allows you to store the resized/compressed images in separate folder, they wil retain the same name.

Force Jpg

This option forces images to be compressed to JPG format.
